Store Operations Assistant

AI-powered guidance for retail operations — Perficient AI Prototype

Situation

Regional managers oversee 8-15 stores. Issues arrive via phone, text, email — no single source of truth. Resolution depends on tribal knowledge. Same problems recur because historical fixes aren't captured.

Task

Build an AI assistant that takes an issue (inventory gaps, staffing crises, compliance) and delivers instant, policy-grounded, data-backed guidance — with full context from the company's own systems.

Action

Built a real-time prototype with two interfaces:

Store Manager Single-store view + AI chat
Regional Manager Multi-store dashboard + AI triage

Agent uses Gemini 3.1 Pro with tool calling to query inventory, issues, policies, and Google Search in a single turn.

Result

Managers get AI-suggested resolutions in seconds vs. waiting for callbacks. Cross-store patterns (e.g., same vendor failing at 4 stores) are detected automatically. Every recommendation cites specific policy IDs and action steps.

Evaluation & Production Roadmap

Evaluation Framework

Golden Dataset

20 scenario-based test queries across store manager & regional manager personas. Each with required references (issue IDs, SKUs, policy IDs) and preferred responses.

Automated Scoring

Reference check: String match for required IDs
Criteria check: Regex for issue IDs, policy refs, action steps, cross-store patterns

LLM Judge

Gemini 3.1 Pro compares actual vs golden response.
Scores 1-5: accuracy, actionability, policy compliance, completeness, groundedness.

Production Data Pipeline

Historical resolutions → human review → curated golden set. Version alongside model/prompt changes. Run evals as CI gate.

Key Decisions & Tradeoffs

Why Convex?

Real-time subscriptions = instant dashboard updates when store managers file issues. Built-in file storage for prototype. Vector search for future semantic retrieval.

Why Gemini 3.1 Pro + Google Search?

Gemini 3 supports built-in tools + function calling in same turn. Google Search grounding adds external context (supplier news, regulations) without a separate search API.

File Storage: R2 for Production

Cloudflare R2 = $0 egress. Retail = high read traffic (managers downloading photos/docs). S3 egress at $0.09/GB compounds across 10+ stores.
